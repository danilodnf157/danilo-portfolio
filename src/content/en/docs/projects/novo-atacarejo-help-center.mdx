    ## Intelligent Support Flow

    The system implements a 9-step wizard that validates each condition necessary for cashback:

    1. **Novo Clube Registration** - Verifies if the user was registered before purchase
    2. **Participating Products** - Confirms if purchased products are on the cashback list
    3. **Purchase Location** - Validates if it was at a physical Novo Atacarejo store
    4. **Checkout Identification** - Confirms if CPF was provided at the POS
    5. **Release Deadline** - Verifies if 48 hours have passed since purchase
    6. **App Consultation** - Guides on how to check the balance
    7. **Balance Validity** - Confirms if the cashback is still valid
    8. **Checkout Usage** - Verifies if the user informed about cashback usage
    9. **Virtual Card** - Confirms if the user has the app in hand

    Each step offers specific guidance and targeted CTAs when necessary.
